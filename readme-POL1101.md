# POL1101 การเมืองและการปกครองของไทย

ข้อมูลวิชาจากวิดีโอที่ประมวลผลแล้วในโปรเจคนี้

## ไฟล์ที่ใช้ทบทวน

- 2026-06-30 / video-01: `classes/POL1101/2026-06-30/video-01/lecture-summary.txt`

## ข้อมูลรายวิชา

- วิชานี้เป็นวิชาพื้นฐานของคณะรัฐศาสตร์
- เนื้อหากว้าง แต่ไม่ได้ลงลึกแบบวิชาเฉพาะขั้นสูง
- ตำราอาจมีข้อมูลบางส่วนที่ไม่อัปเดต อาจารย์จะอธิบายส่วนที่เปลี่ยนแปลงระหว่างเรียน
- ผู้สอนในคาบนี้กล่าวถึงการสอนร่วมกันระหว่าง รศ.ดร.วิทยา จิตนุพงษ์ และ รศ.ดร.สุภัทรา มสวัสดิ์
- จำนวนครั้งเรียนต้องดูตารางจริงของภาคเรียน แต่ควรไม่น้อยกว่า 10 ครั้งตามเกณฑ์ที่อาจารย์กล่าวในคาบ

## โครงสร้างเนื้อหา

- ภาคความนำทั่วไป: การเมือง การปกครอง รูปแบบ ลักษณะ ลัทธิ อำนาจอธิปไตย และประชาธิปไตย
- ภาคประวัติศาสตร์: สุโขทัย อยุธยา ธนบุรี รัตนโกสินทร์ รัฐสมัยใหม่ สยาม/ไทย และพัฒนาการรัฐธรรมนูญ
- ภาคสถาบันทางการเมือง: รัฐธรรมนูญ สถาบันพระมหากษัตริย์ คณะรัฐมนตรี รัฐสภา ตุลาการ พรรคการเมือง กลุ่มผลประโยชน์ และการเลือกตั้ง
- ภาคการบริหารราชการแผ่นดิน: ส่วนกลาง ส่วนภูมิภาค ส่วนท้องถิ่น กรุงเทพมหานคร และเมืองพัทยา

## คะแนนและการสอบ

- อาจารย์ระบุว่าวิชานี้น่าจะมีทั้งข้อสอบปรนัยและอัตนัย หรือเป็นข้อสอบผสม
- รายละเอียดคะแนนเก็บยังไม่สรุปในคาบแรก ต้องรออาจารย์แจ้งอีกครั้งหลังประสานกับผู้สอนอีกท่านและดูตารางวิชาอื่น
- ภาคพิเศษมีแนวโน้มมีคะแนนเก็บ ไม่ได้พึ่งคะแนนสอบ final อย่างเดียว
- ข้อสอบอาจมีคำถามจากข่าวการเมืองปัจจุบัน เช่น ผู้ดำรงตำแหน่งทางการเมืองในเวลาสอบ

## เอกสารและการสื่อสาร

- ควรสร้างกลุ่ม LINE ของ POL1101 ภาคพิเศษ เพื่อใช้รับประกาศรายวิชา
- อาจารย์ต้องการให้มีตัวแทนหรือหัวหน้าห้องคอยประสาน
- ควรดึงอาจารย์และผู้ประสานงานโครงการเข้ากลุ่ม เพื่อประกาศคะแนนเก็บ การเปลี่ยนแปลงคาบเรียน หรือเรื่องเร่งด่วน
- ใน transcript มีการกล่าวถึงคุณ/พี่สมปราชญ์เป็นผู้ประสานงาน และมีหมายเลขโทรศัพท์ของอาจารย์ที่ถอดจากเสียงได้ใกล้เคียง 092-641-4441 ควรตรวจสอบจากประกาศจริงอีกครั้งก่อนใช้เป็นข้อมูลติดต่อถาวร

## สิ่งที่ต้องทำ

- อ่านบทต้นของตำราและดูสารบัญเพื่อเข้าใจโครงสร้างทั้งเล่ม
- ติดตามข่าวการเมืองปัจจุบันควบคู่กับการอ่านตำรา
- เตรียมดินสอ 2B สำหรับข้อสอบปรนัย
- เตรียมปากกาน้ำเงินหรือดำสำหรับข้อสอบอัตนัย
- รอฟังประกาศเรื่องคะแนนเก็บและรูปแบบข้อสอบที่ชัดเจน
- เข้ากลุ่ม LINE รายวิชาเมื่อมีการตั้งกลุ่มแล้ว
