## การเรียนและแนวทางของอาจารย์

- อาจารย์ย้ำตั้งแต่คาบแรกว่าเป้าหมายของการเรียนคือ “เรียนให้รู้เรื่อง” ไม่ใช่เรียนเพื่อให้จบโดยไม่มีความรู้
- วิชานี้ใช้ข้อสอบอัตนัย จึงต้องเขียนคำตอบให้มีเหตุผลและอ่านออก
- อาจารย์ให้ความสำคัญกับการอ่านหนังสือประกอบ เพราะหนังสือช่วยให้เห็นรูปแบบการเขียนเชิงวิชาการและคำอธิบายที่ใช้ตอบข้อสอบได้

## คะแนนและการสอบ

- มีการเก็บคะแนนประมาณ 2 ครั้ง
- การเก็บคะแนนเป็นข้อสอบอัตนัย และช่วงเก็บคะแนนอาจารย์อนุญาตให้เปิดตำราได้
- การสอบประจำวิชาหรือปลายภาคสอบกับโครงการ เป็นการปิดตำรา
- อาจารย์จะบอกแนวให้ว่าให้ไปดูเรื่องใดหรือท่องเรื่องใดมา
- ห้องสอบมีกรรมการคุมสอบและดูแลเรื่องการลอกข้อสอบจริงจัง
- สอบซ่อมทำได้ แต่โดยหลักไม่สามารถได้ A และอาจได้สูงสุดประมาณ C
- ลายมือสำคัญมาก ถ้าเขียนจนอ่านไม่ได้ อาจารย์อาจไม่อ่านและให้ตกได้

## สิ่งที่ต้องระวัง

- อย่าพึ่งสอบซ่อมเป็นแผนหลัก ควรทำคะแนนเก็บและสอบรอบปกติให้ดีที่สุด
- การตอบข้อสอบอัตนัยต้องเขียนให้อ่านออกและตอบเป็นเรื่องเป็นราว
- อาจารย์ไม่สนับสนุนการลอกข้อสอบหรือการเรียนแบบ “จ่ายครบจบแน่”
