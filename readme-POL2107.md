- อาจารย์มองว่าไม่ใช่วิชาง่ายมาก แต่ก็ไม่ยากเกินไปถ้าฟังตามลำดับและเข้าใจภาพรวมก่อน

## คะแนนและกิจกรรม

- อาจารย์พูดถึงคะแนนพิเศษจากการถามคำถามในชั้นเรียน
- ตัวอย่างที่อาจารย์ยกคือ ถามคำถามแล้วได้ 5 คะแนน ถ้าถามได้ 4 ครั้งอาจสะสมได้ถึง 20 คะแนน
- คะแนนพิเศษสามารถนำไปบวกกับคะแนนสอบได้
- วิชานี้มีนักศึกษาจำนวนมาก อาจารย์คาดว่าน่าจะใช้ข้อสอบปรนัยหรือ multiple choice เป็นหลัก
