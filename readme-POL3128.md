# POL3128

## ข้อมูลรายวิชา

- รหัสวิชา: POL 3128
- ชื่อวิชา: แนววิเคราะห์ทฤษฎีการเมือง
- ผู้บรรยาย: อาจารย์ศักดิ์ราช เจริญทรัพย์
- รูปแบบการเรียน: ฟังคำบรรยายออนไลน์

## เอกสารประกอบ

อาจารย์แจ้งว่าจะพยายามค้นหาและแนะนำเอกสารให้นักศึกษาอ่านเพิ่มเติม โดยอาจเป็นตำราของมหาวิทยาลัยรามคำแหง หนังสือที่ซื้อได้ทั่วไป หรือไฟล์ PDF ที่อ่านได้จากเว็บไซต์ หากเป็นไปได้ โครงการจะจัดเอกสารประกอบให้นักศึกษาเข้าถึงผ่าน QR Code

## การทบทวนคำบรรยาย

อาจารย์แนะนำให้เปิดฟังคำบรรยายหลายครั้ง หากฟังครั้งแรกแล้วหลุดประเด็นหรือยังไม่เข้าใจ ให้ย้อนกลับไปฟังซ้ำ

## การติดต่อและคำถาม

หากมีคำถามจากการฟังคำบรรยายออนไลน์ นักศึกษาสามารถฝากคำถามไว้กับผู้ประสานงานหรือโครงการ เพื่อให้อาจารย์รับไปตอบเมื่อมีเวลา

หากนักศึกษาต้องการพบอาจารย์เป็นกลุ่มเพื่อสนทนาและแลกเปลี่ยนความเห็น ให้ประสานกับทางโครงการ อาจารย์แจ้งว่ายินดีจัดเวลาให้เท่าที่ทำได้

## หัวข้อครั้งถัดไป

อาจารย์จะเริ่มอธิบายหลักการและองค์ประกอบที่ใช้ศึกษาค้นหาความเป็นเหตุเป็นผลของสิ่งต่าง ๆ
